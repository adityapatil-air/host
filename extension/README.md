# Floating Translator Extension

A cross-platform floating translator that stays on top of all applications.

## Quick Start

### Option 1: Auto Setup
```bash
python setup.py
```

### Option 2: Manual Setup

#### Windows
```cmd
pip install requests pyperclip Pillow
python floating_translator.py
```

#### macOS/Linux
```bash
pip3 install requests pyperclip Pillow
python3 floating_translator.py
```
